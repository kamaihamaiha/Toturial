## 2. 属性动画(Property Animation)
### 2.1 简介
- 作用对象： 任意 Java 对象
- 动画效果： 可以自定义

### 2.2 工作原理
- 在一定时间间隔内，不断对值进行改变，同时把该变化的值赋值给对象的属性，从而实现该对象在该属性上的动画效果
- 2 个重要的类： ValueAnimator, ObjectAnimator
