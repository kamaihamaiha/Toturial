## 动画

- 视图动画 = 补间动画 + 逐帧动画
- 属性动画
### 补间动画
#### 1. 作用对象
视图空间 (View)
```
不能作用于 View 的属性，如： 颜色、背景、长度   
```

#### 2. 原理
确定开始视图样式和结束视图样式，中间动画变化过程有系统补全。

#### 3. 分类
- 平移 (Translate)
- 缩放 (scale)
- 旋转 (rotate)
- 渐进 (alpha)
